## Summary

If you open the `slnx` file in Visual Studio, Rider or any other tool, you just need to:

1. Add an Application Insights Connection string to the `properties/launchSettings.json`.
2. Uncomment the lines in the `Program.cs`.

If your Application Insights component is configured with disabled local auth you can only send data to it, if you are logged in correctly.

Watch the video to get more details.
