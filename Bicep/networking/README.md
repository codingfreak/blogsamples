# Youtube Playlist `Bicep: Azure Networking`

The goal of the YT series is to show how to use Bicep and related tools to define
network resources in Azure. Beginning with basic topics like network design I slowly
want to move towards more complicated topics like gateways, VPNs, DNS and the like.
